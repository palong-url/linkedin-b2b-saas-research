# LinkedIn Organic Content Strategy for B2B SaaS — Research

Research project analyzing how top practitioners grow B2B SaaS businesses using LinkedIn organic content.

## Why this topic

LinkedIn organic is one of the highest-ROI channels for B2B SaaS — no ad spend required, long shelf life per post, and direct access to decision-makers. The challenge is most advice is generic. This research collects content from people who have actually built pipeline and revenue from it.

## How content was collected

- **LinkedIn posts:** Manually collected from each expert's profile (latest 5–10 posts)
- **YouTube transcripts:** Downloaded via `youtube-transcript-api` Python library
- **Other:** Newsletter archives, publicly available reports
